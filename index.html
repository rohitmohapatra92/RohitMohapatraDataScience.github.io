My Projects

Heart Disease Risk Predictor

This project is built on Jupyter notebook to tackle a classification scenario in order to predict if someone is at a risk of having a heart disease. In this project I have built various models using Logistic Regression, Random Forest using GridSearchCV, XGBoost and also a Neural Network.

Customer Segmentation

This project aims to drive the marketing campaign by targetting specific customers. I have used the K-Means clustering algorithmn to segment customers that aims to help the marketing team to analyze which customer segment is most likely to buy the product and then market the product only on that particular segment.

Laptop Price Predictor

This project is heavy on data cleaning and pre-processing and finally uses regression techniques like Linear and Ridge Regression, Random Forest Regressor, KNN, Gradient Boosting and XGB Regressor. This project also uses tools like Column transformers and Pipelines.

Analysis of Deaths Involving Police

This project analyzes database as compiled by The Washington Post of every fatal shooting in the US by a police officer in the line of duty. The project demonstrates skills such as data cleaning and visualization techniques using Matplotlib, Seaborn and Plotly Express and is built on Jupyter Notebook.

Analysis of the Space Race

This project uses the dataset that was scraped from nextspaceflight.com and includes all the space missions since the beginning of Space Race between the USA and the Soviet Union in 1957. It demonstrates skills such as data cleaning, EDA and visualization using Matplotlib, Seaborn and Plotly Express

IPL Performance Analysis

This is a SQL project built on MySQL to answer questions and drive insights from data that includes performances and statistics of players playing in the IPL. The project demonstrates skills like writing queries, grouping and joing relational databases.
